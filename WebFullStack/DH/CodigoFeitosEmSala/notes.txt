#Extenssões
emmet -> extensão (deafult) do visual studio code que cria estruturas de codigos completos com pequenas linhas de código
beautify -> extensão que identa o codigo (press CTRL+SHIFT+P e digite beautify file)
live server -> abre seu projeto direto no browser (ALT+L+O)
